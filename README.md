# Alfavet

Next.js homepage redesign for Alfavet veterinary clinic.

## Stack

- Next.js App Router
- React
- TypeScript
- CSS Modules
- lucide-react icons

## Commands

```bash
pnpm dev
pnpm lint
pnpm build
```

The current homepage includes light/dark themes, AZ/EN/RU copy, reusable UI buttons, a real Alfavet logo, partner assets and the clinic hero video.
